  • Volg de instructies!!!!!

1. Student nummer aanpassen

Opdracht

Voor je begint met het aanpassen van de code, zorg dat je in de main branch zit.

  • Vervang bovenaan de pagina "Student nummer van student" door jouw eigen student nummer (id=student-nummer)
  • Commit deze wijziging

2. Afbeeldingen toevoegen

Opdracht

Voor je begint met het aanpassen van de code, zorg dat je in de main branch zit.

  • Maak een nieuwe branch: afbeeldingen
  • Controleer dat je in deze branch zit
  • Zoek een passende afbeelding voor dit hoofdstuk (id=afbeeldingen), voeg deze afbeelding toe in de map images.
  • Pas de src aan van de afbeelding van dit hoofdstuk aan, zodat de nieuw toegevoegde afbeelding zichtbaar is.
  • Commit deze wijziging
  • Ga terug naar de main branch.
  • Merge de aanpassingen van afbeeldingen naar main.
  • Push de main branch naar github

3. Een conflict oplossen

Deadpool

Deadpool

  • Zorg dat je in de main branch zit
  • Begin een nieuwe branch: Deadpool
  • Pas de hierboven aangegeven winnaar (id=winnaar) aan naar: Deadpool
  • Commit deze wijziging.

Punisher

  • Ga terug naar de main branch
  • Begin een nieuwe branch: punisher
  • Pas de hierboven aangegeven winnaar (id=winnaar) aan naar: Punisher
  • Commit deze wijziging.

Conflict!

  • Ga terug naar de main branch
  • Refresh de pagina en check of Niemand de winnaar is
  • Merge de aanpassingen van punisher naar main.
  • Controleer of punisher de winnaar is (refresh de pagina).
  • Merge de aanpassingen van deadpool naar main.
  • Conflict!

    Je hebt nu onderstaand conflict in de header van dit article.

    
<<<<<<< HEAD
<span id="winnaar">Punisher</span>
=======
<span id="winnaar">Deadpool</span>
>>>>>>> Deadpool
  • Zorg dat Deadpool de winnaar is.
  • Commit de merge
  • Push de main branch naar github

4. Een conflict oplossen

Soms is er bij een conflict geen keuze, maar moet je handmatig de wijzigingen samen voegen.

Schadevergoeding: € 2.345.999.999,00

Avengers

  • Zorg dat je in de main branch zit
  • Begin een nieuwe branch: Avengers
  • Pas het hierboven aangegeven schadevergoedingbedrag (id=schadevergoeding) aan naar: 2.000.000,00
  • Commit deze wijziging.

Savage Avengers

  • Ga terug naar de main branch
  • Begin een nieuwe branch: Savage Avengers
  • Pas het hierboven aangegeven schadevergoedingbedrag (id=schadevergoeding) aan naar: 2.345.999.999,00
  • Commit deze wijziging.

Conflict!

  • Ga terug naar de main branch
  • Merge de aanpassingen van Savage Avengers naar main.
  • Controleer of de schadevergoeding 2.345.999.999,00 is (refresh de pagina).
  • Merge de aanpassingen van Avengers naar main.
  • Conflict!

    Je hebt nu onderstaand conflict in de header van dit article.

    
<<<<<<< HEAD
<span id="donaties">€ 2.345.999.999,00</span>
=======
<span id="donaties">€ 2.000.000,00</span>
>>>>>>> Avengers
  • Zorg dat de schadevergoedingen bij elkaar worden opgeteld.
  • Controleer of het schadevergoedingbedrag correct is (refresh de pagina).
  • Commit de merge
  • Push de main branch naar github

5. Versie update

Versiebeheer eindigt natuurlijk altijd met een nieuwe versie.

Opdracht

Voor je begint met het aanpassen van de code, zorg dat je in de main branch zit.

  • Verhoog de versie (id=version) met 0.1
  • Commit en push de wijziging
  • Lever de URL van jouw github repository in.
